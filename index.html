História dos computadores

A primeira geração de computadores (1951-1959) funcionava a partir de válvulas eletrônicas, o que gerava equipamentos gigantescos e que consumiam muita energia.

A segunda geração de computadores (1959-1965) funcionava a partir de transistores, o que gerava equipamentos menores que o da primeira geração, mas ainda assim grandes. Com estas características, iniciou-se o uso de computadores para fins comerciais.

A terceira geração de computadores (1965-1975) funcionava a partir de circuitos integrados. A diminuição dos tamanhos dos equipamentos permitiu o surgimento dos computadores pessoais.

A quarta geração de computadores (1975 ao presente) é marcada pelo desenvolvimento de computadores cada vez menores, com maior velocidade e capacidade de processamento de dados. A partir dos anos 1990, houve uma maior popularização dos computadores pessoais.

Alguns pesquisadores defendem a existência de uma quinta geração de computadores que decorreria do surgimento dos supercomputadores utilizados por grandes corporações.

História da Internet

Com o objetivo de compartilhamento de informações de interesse estratégico-militar, o governo dos Estados Unidos da América criou um sistema de compartilhamento de dados denominado Arpanet. O desenvolvimento da Arpanet gerou a Internet.

Em 1969 ocorreu a primeira transmissão de dados da história, entre duas universidades dos EUA. O texto que se desejava enviar era LOGIN, mas apenas LO foi recebido.

O sistema técnico TCP/IP permitiu a comunicação entre computadores a partir de uma padronização necessária. Décadas depois, a união do TCP/IP com hipertexto criou a World Wide Web (WWW).
